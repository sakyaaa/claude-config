# Правила для репозитория claude-config (~/.claude/)

Этот каталог — git-репозиторий с глобальным конфигом Claude Code.
Remote: git@github.com:sakyaaa/claude-config.git (ветка main)

## После каждого изменения

1. Обнови `MEMORY.md` — добавь запись о том, что изменилось и зачем
2. Сделай коммит: `git commit -m "Краткое описание изменения"`
3. **Обязательно запушь**: `git push` — конфиг должен быть синхронизирован с remote

## Что отслеживается

| Файл | Назначение |
|------|-----------|
| `CLAUDE.md` | Глобальные инструкции для Claude (грузится в каждый промпт) |
| `MEMORY.md` | История изменений конфига |
| `settings.json` | Настройки Claude Code: разрешения, статус-бар |
| `statusline.sh` | Скрипт кастомного статус-бара |
| `.gitignore` | Исключения: приватные данные, кэш, сессии |
| `.claude/CLAUDE.md` | Этот файл — project-level инструкции только для этого репо |

## Что НЕ отслеживается (.gitignore)

- `.credentials.json` — токены и секреты
- `history.jsonl` — история сессий
- `cache/`, `sessions/`, `plans/`, `plugins/` и прочие временные каталоги
