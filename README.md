# claude-config

Персональный конфиг [Claude Code](https://claude.ai/claude-code) — глобальные настройки, инструкции и кастомизации, которые применяются во всех проектах.

## Что это такое

Claude Code хранит глобальный конфиг в директории `~/.claude/`. Этот репозиторий версионирует ту часть конфига, которую имеет смысл хранить: инструкции для Claude, настройки разрешений, кастомный статус-бар.

## Структура

```
~/.claude/
├── CLAUDE.md           # Глобальные инструкции для Claude (загружаются в каждую сессию)
├── MEMORY.md           # История изменений конфига
├── settings.json       # Настройки Claude Code (разрешения инструментов, статус-бар)
├── statusline.sh       # Скрипт кастомного статус-бара
├── .gitignore          # Исключения: секреты, кэш, временные файлы
└── .claude/
    └── CLAUDE.md       # Инструкции для Claude при работе именно в этом репо
```

## Возможности

### Статус-бар

Кастомный статус-бар в нижней части терминала отображает:
- Текущую модель Claude
- Git-ветку рабочего проекта
- Процент использования контекстного окна
- Время текущей сессии

### Глобальные инструкции (CLAUDE.md)

Определяют поведение Claude во всех проектах: язык ответов (русский), структуру файлов при инициализации проекта, обязательные действия после выполнения задач, формат коммитов и веток.

### Разрешения (settings.json)

Настроено автоматическое разрешение для `WebSearch` и `WebFetch` — поиск в интернете не требует подтверждения каждый раз.

## Установка на новую машину

```bash
git clone git@github.com:sakyaaa/claude-config.git ~/.claude
```

> **Важно:** `.credentials.json` не версионируется — его нужно получить отдельно или пройти авторизацию заново через `claude`.
